# dotnet-web-learning

**Create Project:**<br/>
```bash
dotnet new webapi -n MyStartApi
```
Go to created project... `MyStartApi`

**Build:**<br/>
```bash
dotnet build
```

**Run:**<br/>
```bash
# normal
dotnet run
# ..or in hot reload mode
dotnet watch run
```

**Create basic gitignore file:**<br/>
```bash
dotnet new gitignore
```
